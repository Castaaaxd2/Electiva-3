========================================
  BirdLens — Dependencias del proyecto
========================================

REQUISITO PREVIO
----------------
Node.js >= 20
pnpm >= 9

  Instalar pnpm (si no lo tienes):
  npm install -g pnpm

  Instalar todas las dependencias del proyecto:
  pnpm install

========================================
  VARIABLES DE ENTORNO REQUERIDAS
========================================

API Server:
  DATABASE_URL          — Cadena de conexión a PostgreSQL
  SESSION_SECRET        — Secreto para sesiones
  AI_INTEGRATIONS_OPENAI_BASE_URL  — Auto-provisto por Replit AI Integrations
  AI_INTEGRATIONS_OPENAI_API_KEY   — Auto-provisto por Replit AI Integrations

  Opcional (modelo personalizado):
  MODEL_SERVER_URL      — URL del servidor Python con el modelo entrenado
                          Ejemplo: http://localhost:5001


========================================
  COMANDOS ÚTILES
========================================

  pnpm install                          Instalar todas las dependencias
  pnpm --filter @workspace/mobile run dev       Iniciar app móvil (Expo)
  pnpm --filter @workspace/api-server run dev   Iniciar servidor API
  pnpm run typecheck                    Verificar tipos TypeScript
  pnpm run build                        Build completo del proyecto
  pnpm --filter @workspace/db run push  Aplicar cambios al esquema de BD

  Expo Go (para probar en físico):
    1. Instalar "Expo Go" desde App Store (iOS) o Play Store (Android)
    2. Escanear el QR que aparece al ejecutar el dev server
